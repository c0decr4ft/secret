# Secret Chat

A hidden chat app disguised as a GitHub 404 page. Type "liath" to unlock.

## Run locally

```bash
npm install
npm start
```

Open **http://localhost:3000** in your browser.

- Type "liath" on the 404 screen to unlock
- Pick your account (Luca, Matteo, or Nora)
- Chat with the other two people
- Messages are saved and sync in real time

## Use on different devices

**Same WiFi:** Open `http://YOUR_COMPUTER_IP:3000` on your phone (find your IP with `ipconfig` or `ifconfig`).

**From anywhere:** Deploy the server to [Railway](https://railway.app) or [Render](https://render.com) (free tier), then use the deployed URL.
